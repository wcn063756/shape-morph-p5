# 几何图形拓扑变换

一个使用p5.js实现的几何图形拓扑变换程序，支持多种几何图形之间的平滑过渡效果。

## 功能特点

- 支持8种几何图形：三角形、正方形、圆形、五边形、六边形、五角星、心形、菱形
- 所有图形都由边框线条组成，没有填充色
- 平滑的拓扑变换动画
- 颜色在变换过程中自动过渡
- 只能向右拖动，图形向正右方移动
- 最终停留在右侧图形上

## 如何运行

1. 访问 [p5.js Web Editor](https://editor.p5js.org/)
2. 创建新的Sketch
3. 复制 `shape-morph-p5.js` 中的代码到编辑器
4. 点击 "Run" 按钮开始运行

## 使用说明

1. 在选择界面中，点击左侧和右侧的选择器
2. 从下拉菜单中选择你想要的几何图形
3. 点击"开始变形"按钮
4. 按住左侧图形向右拖动，观察拓扑变换效果

## 支持的图形

- 三角形（珊瑚红）
- 正方形（道奇蓝）
- 圆形（金色）
- 五边形（深海洋绿）
- 六边形（深兰花紫）
- 五角星（橙红色）
- 心形（红色）
- 菱形（矢车菊蓝）
